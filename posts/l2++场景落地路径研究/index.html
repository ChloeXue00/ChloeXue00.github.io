二、典型场景落地路径分析#

1️ 高速场景：最成熟的L2++落地入口#

路径演进：

  • 从ACC/LKA演进到NOA
  • 配合高精地图完成匝道上下、变道超车
  • 再演进为“无图NOA”

代表功能：

  • 自动变道超车（ALC）
  • 自动上下匝道
  • 智能限速

代表落地品牌：
蔚来NOP+、华为ADS、Momenta Mpilot、理想NOA

2️ 城市道路场景：能力挑战最大、战略重点场景#

路径演进：

  • 从城市主干道直行 → 红绿灯识别通行
  • → 支持交叉路口通行、无保护左转、避让行人等
  • → BEV感知+轻地图+规则策略并存

代表功能：

  • 城市NOA
  • 红绿灯识别与响应
  • 异形障碍避让
  • 非保护转弯/礼让行人/加塞识别

代表落地品牌：
Momenta、华为ADS 2.0、百度Apollo Max、文远WePilot

3️ 泊车类场景：感知主导场景、落地稳定#

路径演进：

  • APA → RPA → 记忆泊车 → 云图泊车

代表功能：

  • 自动泊车（APA）
  • 远程控制泊车（RPA）
  • 记忆泊车（VPA）

代表落地品牌：
蔚来APA、比亚迪DiPilot、上汽智己APA+、理想记忆泊车

4 通勤类场景：用户粘性强的私域路径#

路径演进：

  • 用户固定路线记忆学习
  • 端到端策略模型持续训练
  • 地图自由化路径探索

代表功能：

  • 通勤NOA
  • 用户驾驶偏好学习（并道习惯、速度设定等）

代表落地品牌：
Momenta通勤模式、百度Apollo通勤助手、理想自定义通勤

5️ 高速+城市融合场景：全域智能驾驶终极形态#

路径演进：

  • 城市NOA + 高速NOA 融合 → 点到点导航辅助驾驶
  • 全场景策略统一、多任务协同调度

代表功能：

  • Point-to-Point Assist（PPA）
  • 从家门口到停车位全过程导航驾驶

代表落地品牌：
极越01、问界M9、智己L6、百度Apollo Max

三、产品规划建议：按“能力演进 + 城市开通”分阶段部署#

阶段 功能重点 依赖 用户价值
第一阶段 高速NOA全量+APA 高精地图（高速） 通勤高速、长途频繁使用
第二阶段 城市快速路+红绿灯通行 轻地图 + 感知能力 日常代驾体验升级
第三阶段 异形障碍、无保护转弯 BEV感知 + 策略学习 抢占长尾能力口碑
第四阶段 熟路线记忆NOA 端到端模型 + 用户数据 用户粘性 + 体验差异化
第五阶段 全域点到点NOA 多策略融合 + OTA升级 品牌“高阶智驾”标签

四、总结#

L2++落地不再是“堆硬件”，而是围绕：

  • 对真实场景的理解
  • 对边界工况的覆盖
  • 对策略融合能力的稳定性
  • 对用户可感知体验提升

进行系统规划与迭代。

产品经理在规划过程中应把握两条主线：

  • 能力演进节奏（从高速 → 城市 → 全域）
  • 城市开放节奏（从一线城市 → 固定通勤 → 全国覆盖）

落地路径应由用户场景反推产品能力模块，形成完整可视化路线图。
